# Financial Transactions EDA & ML

This project analyzes financial transaction data and builds machine learning models to predict transaction success or failure. It includes data cleaning, exploratory data analysis (EDA), feature encoding, and model evaluation.

## Folder Structure

Financial_Transactions_EDA/
├── data/         ← CSV file(s) with financial transaction data
│     └── dirty_financial_transactions.csv
├── notebooks/    ← Jupyter notebook(s) with EDA + ML code
│     └── ml_models.ipynb
├── scripts/      ← Optional helper Python scripts (can be empty)
└── README.md     ← This README file

## How to Run

1. Clone the repo
2. Make sure the CSV file is in `data/`
3. Open the notebook in `notebooks/` folder
4. Run all cells

## Libraries

- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn
